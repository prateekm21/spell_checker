## Dependencies 

```
Node.js
npm install -g mocha
gem install  rake
```

## RAKE

rake \<return\>

```
rake init                : npm install: download all dependencies.
rake run                 : run the spell checker
rake spec                : run unit tests
rake spec_run            : run unit tests followed by spell checker
```

## Specs

    rake spec
    
## Start SpellChecker

    rake spec_run
    rake run
